## Why Phase 0 Matters (The A/B Test)

The single most important question in this entire project: **Does multi-agent beat single-agent for traction inverter design?**

Phase 0 answers this by building BOTH:
1. **Single-agent baseline:** One well-tooled agent with all tools
2. **Minimal multi-agent:** 3 agents (Orchestrator, Simulation, Reviewer) — the simplest possible decomposition

If single-agent ≥ multi-agent on the benchmark: **Stop. Simplify. Ship the single agent.**

If multi-agent > single-agent: **Proceed to Phase 1 with evidence.**

No other phase matters until this question is answered.

### P0.2 — SPICE Template Library (Days 1-2)

**Why templates not raw LLM generation:** HAVEN (2026) showed template-based generation prevents hallucination in HDL. SPICE is similarly syntax-sensitive. One wrong node number = broken simulation. LLMs fill template parameters; they don't write raw SPICE.


### P0.3 — Single Agent Loop (Days 2-4)

```python
# src/srtp_ai/agent.py
from liteLLM import completion
from pydantic import BaseModel

class InverterSpec(BaseModel):
    vdc: float          # DC-link voltage [V]
    pout: float         # Rated power [kW]
    topology: str       # "2L-B6" | "3L-NPC"
    efficiency_target: float  # e.g., 0.97

class SimulationResult(BaseModel):
    efficiency: float
    thd: float
    vds_max: float
    tj_est: float
    converged: bool
    waveforms: dict     # {time: [...], va: [...], ia: [...]}

class SingleAgent:
    """Single well-tooled agent for traction inverter design."""
    
    def design(self, spec: InverterSpec) -> dict:
        """Design an inverter from specification."""
        # 1. Select topology (if not specified)
        topology = self._select_topology(spec)
        
        # 2. Size components
        components = self._size_components(spec, topology)
        
        # 3. Fill SPICE template
        netlist = self._fill_template(topology, components, spec)
        
        # 4. Run simulation
        result = self._simulate(netlist)
        
        # 5. Analyze results
        analysis = self._analyze(result, spec)
        
        # 6. Iterate if needed
        if analysis.efficiency < spec.efficiency_target:
            components = self._refine(spec, components, analysis)
            # ... loop back to step 3
        
        return {"spec": spec, "components": components, "result": result, "analysis": analysis}
```

**LLM calls in this loop:**
1. `_select_topology()` — 1 LLM call (choose topology for spec)
2. `_size_components()` — 1 LLM call (calculate L, C, fs, device ratings)
3. `_fill_template()` — 0 LLM calls (deterministic parameter substitution)
4. `_analyze()` — 1 LLM call (interpret simulation results vs spec)
5. `_refine()` — 1 LLM call (adjust parameters)

**Total: 4 LLM calls per iteration.** At ~$0.01/deepseek-chat call: $0.04/iteration.

**Deliverable:** `SingleAgent.design(spec)` returns complete design with simulation results.
**Verify:** Design a 400V 150kW 2L-B6 inverter. Simulation must converge. Efficiency > 90% (low bar for now).

### P0.5 — Minimal Multi-Agent (Days 6-8)

**3 agents only** — the simplest useful decomposition:

```python
from langgraph.graph import StateGraph, END
from langgraph.checkpoint.sqlite import SqliteSaver

class OrchestratorAgent:
    """Decomposes spec → delegates to Sim Agent → collects results."""
    def plan(self, spec: InverterSpec) -> SimulationPlan: ...
    def decide(self, result: SimulationResult, spec: InverterSpec) -> str:  # "iterate" | "done"

class SimulationAgent:
    """Fills SPICE templates, runs PySpice, returns structured results."""
    def run(self, plan: SimulationPlan) -> SimulationResult: ...

class ReviewerAgent:
    """Checks results against spec, identifies issues, suggests fixes."""
    def review(self, result: SimulationResult, spec: InverterSpec) -> ReviewReport: ...

# LangGraph state machine
class DesignState(TypedDict):
    spec: InverterSpec
    plan: SimulationPlan | None
    components: dict | None
    netlist: str | None
    result: SimulationResult | None
    review: ReviewReport | None
    iteration: int
    converged: bool

workflow = StateGraph(DesignState)
workflow.add_node("orchestrator", orchestrate)
workflow.add_node("simulate", simulate)
workflow.add_node("review", review)
workflow.add_conditional_edges("review", should_continue, {
    "iterate": "orchestrator",
    "done": END
})
workflow.set_entry_point("orchestrator")

checkpointer = SqliteSaver.from_conn_string("checkpoints.db")
app = workflow.compile(checkpointer=checkpointer)
```

**Deliverable:** 3-agent LangGraph system that completes the same design task as the single agent.
**Verify:** Run against benchmark. Compare results.

```python
def validate_design(result: SimulationResult, spec: InverterSpec) -> list[str]:
    violations = []
    
    # G1: Thermal safety
    Tj_max = 175 if "SiC" in spec.topology else 150
    if result.tj_est > Tj_max - 25:
        violations.append(f"Tj={result.tj_est}°C exceeds {Tj_max-25}°C margin")
    
    # G2: Voltage margins
    if result.vds_max > 0.8 * spec.vdc:
        violations.append(f"Vds_max={result.vds_max}V exceeds 80% rating")
    
    # G3: Physical realism
    if result.efficiency > 0.999:
```
