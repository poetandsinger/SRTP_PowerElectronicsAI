# knowledge/papers/ — raw source PDFs (read-only)

Raw paper/datasheet PDFs go here, one per source, **never edited** (read-only zone,
see [../SCHEMA.md](../SCHEMA.md) §Repository Organization). Name each to match its
digested note: `papers/<field>/foo.pdf` ↔ `notes/<field>/foo.md`.

**Currently empty.** As of the 2026-07-20 reorg this project captures sources as
**markdown** in [`../sources/`](../sources/) (`type: source`), linked to notes by a
`sources:` frontmatter field rather than by filename. The vendor manual
`PLECS_Model_User_Guide_Rev2.pdf` lives with its model library under
`system/env/models/wolfspeed/`, not here. Add raw PDFs here as they are collected.
