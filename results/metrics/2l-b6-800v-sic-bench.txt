2L-B6 800V SiC CAB450 bench — PLECS validation results (2026-07-21)
====================================================================
Model : experiments/2l-b6-800v-sic-bench/bench_2l_b6_800v_sic.plecs
Device: Wolfspeed CAB450M12XM3 (file:CAB450M12XM3.xml, Rgon=4/Rgoff=0 = datasheet-nominal Eon/Eoff)
Solver: dopri variable-step, RelTol 1e-4, MaxStep 1e-4, ZCStepSize 1e-9; fs=16kHz, fac=200Hz, TimeSpan 0.05s (10 fund. periods)
Controller: SV PWM (extended linear range); filter Lg = 0.5 p.u. reactance (motor-like, clean current)
Readback: every metric via ToFile->CSV (simulate Values empty in 4.8); steady-state = last integer # of fundamental cycles.

Loss chain: conduction = PeriodicAverage(T=1/fac) on all 6; switching = PeriodicImpulseAverage(T=1/fac) on all 6.
Efficiency: eta = (Pin - Ploss)/Pin, Pin = Vdc*Idc, Ploss = Sigma6(conduction+switching).
Energy balance (S3): residual = Pin - Ploss - (P_into_backEMF + P_Rg), independent AC-side measurement.
Analytic Tj_ss (option b, thermal timescale >> 0.05s switched run): Tj = Ta + P_module*R_cs + P_switch*R_jc,
  R_jc = 0.0948 C/W (XML Cauer sum, per switch), R_cs = 0.070 C/W/module (CRD-calibrated: puts rated 360A/800V at Tj=175C;
  app-note [168] gives ~0.08 -> Tj~185C, pessimistic). In-model transient Tj readout ("Device junction temp") also works.

  Analytic conduction (2L-VSI closed form, per switch): Ron*Ipk^2*(1/8 + m*cosphi/(3pi))
    = 0.0036 * 508.7^2 * 0.2311 = 215.3 W  vs  PLECS 207.7 W  ->  -3.5% (within +/-10%). PASS.
  Switching loss: from the authoritative Wolfspeed XML 4-D Eon/Eoff(V,I,Tj) tables (more accurate than closed form).
  numpy experiments/ models (family-car, performance, truck) use invented/class-typical device params (status unverified);
    directional only. This PLECS-with-loss-tables result is the datasheet-backed number those hypotheses awaited.

TRENDS (physical sanity)
  eta rises as load falls (99.07 -> 99.32%); switching loss scales with bus (1450/1569/1688 W at 750/800/850V);
  conduction ~flat with bus, scales with I^2 (1246 -> 841 -> 288 W at 360/300/180 A); current clean (crest 1.46, THD 0.15%).
